# 📊 E-Commerce Order Analysis Dashboard

An interactive **Power BI** dashboard developed to analyze E-Commerce Order data and provide meaningful business insights through dynamic visualizations.

## 🚀 Project Overview

This dashboard helps stakeholders monitor sales performance, evaluate sales teams, and analyze customer orders using interactive reports and filters.

## 📑 Dashboard Pages

### 1. Sales Team
- Agent-wise sales performance
- Individual sales comparison
- Sales KPIs by agent

### 2. Sales Manager
- Sales manager performance analysis
- Team-wise sales overview
- Manager performance tracking

### 3. Orders
- Customer order analysis
- Orders by Category
- Orders by Country
- Orders by Sales Manager
- Interactive filtering and drill-down

## ✨ Key Features

- Interactive dashboards
- Dynamic slicers and filters
- KPI Cards
- Drill-down analysis
- Business-friendly visualizations
- Clean and responsive report design

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Data Modeling

## 🎯 Objective

To transform raw E-Commerce order data into an interactive business intelligence dashboard that enables data-driven decision-making.
